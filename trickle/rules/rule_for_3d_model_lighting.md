When user reports model over-exposure or lighting issues in Vision Pro showcase
- Reduce exposure value from default to 0.8 or lower for proper lighting
- Use neutral tone-mapping instead of commerce for more natural appearance
- Set shadow-intensity to 0.4 for subtle, realistic shadows
- Increase shadow-softness to 0.8 for smooth shadow edges
- Reduce environment-intensity to 0.7 to prevent over-bright reflections
- Use neutral environment-image for consistent lighting conditions
- Applied these settings to fix Vision Pro model over-exposure issues
- Always update README.md when lighting settings are modified
